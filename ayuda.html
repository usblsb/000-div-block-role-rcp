Ayuda para el Plugin: Simple Div Block Role y Capabilities

¿Qué hace este plugin?

Este plugin añade un nuevo bloque 'div' en el editor Gutenberg de WordPress, el cual se muestra en el frontend solo a usuarios con determinados roles, capacidades y un estado de membresía activa en Restrict Content Pro.

Beneficios del Plugin

El plugin es útil para sitios web que requieren mostrar contenido exclusivo basado en roles de usuario, capacidades específicas y membresías activas. Esto permite a los administradores de sitios web controlar quién puede ver ciertos contenidos, lo que es ideal para sitios de membresía, contenido premium o áreas exclusivas para ciertos usuarios.

¿Cómo se usa?

Para utilizar este bloque, sigue estos pasos:

  1. Abre el editor Gutenberg en tu sitio WordPress.
  2. Busca y añade el bloque 'Simple Div Block' en tu página o entrada.
  3. En las opciones del bloque, establece los parámetros deseados para 'Role', 'Capabilities' y verifica si el usuario debe tener una membresía activa.
  4. El bloque 'div' solo se mostrará a los usuarios que cumplan con estos criterios específicos en el frontend.

Ejemplo de uso en Gutenberg:

  • Añadir el bloque 'Simple Div Block' a una página.
  • Configurar el bloque para que sea visible solo por usuarios con el rol de 'editor' y la capacidad de 'edit_posts'.
  • El contenido dentro de este bloque solo será visible para usuarios logueados que cumplan con estos criterios.

Nota: Este plugin requiere que Restrict Content Pro esté activo y configurado en tu sitio WordPress para la funcionalidad relacionada con las membresías.
